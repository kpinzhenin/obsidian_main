Можно сохранить консольный вывод GDB в файл. Для этого после запуска вводим команду:
```bash
(gdb) set logging enabled on
```
при этом создастся файл gdb.txt в котором вся история отладки будет.
Если хочется поменять название файла можно использовать команду
```bash
(gdb) set logging file <file_name>
```
В листинге лога можно оставлять комментарии в строке, начинающейся с `#`

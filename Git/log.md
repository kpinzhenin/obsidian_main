`git log --name-status` - посмотреть файлы которые добавлены в коммит
Посмотреть номер коммита и комментарий к коммиту только
```bash
git log --oneline
```
Посмотреть в виде дерева с указанием номера и комментария к коммиту
```bash
git log --graph --oneline
```
Все файлы, когда-либо существовавшие в репозитории
```bash
git log --pretty=format: --name-only | sort -u
```

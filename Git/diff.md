Выводит разницу между коммитами/файлами.
разница между 
```bash
git diff <commit_name>
```
Возвращает разницу между текущей версией ( не индексированной ) и тем, что указано в аргументе ( в данном случае коммите ).
```bash
git diff <branch_name> --name-only
```
возвращает разницу между ветками - выводит список отличающихся файлов (`--name-only`)

По сути оптимизирует ( снижает ) количество `move` инструкций в коде.
Например для кода 
```c
int foo(int a, int b) {
    int c = a;
    return c + b;
}
```
сформируется ассемблерный код:
```asm
mov eax, edi  ; копирование a в c (регистр eax)
add eax, esi
```
который будет оптимизирован в:
```
lea eax, [edi + esi]
```

Существуют несколько подходов в организации структуры проекта.
### Modern approach
Каждый "модуль" ( грубо говоря - каждая папка с исходниками собирается своим ( вложенным в эту папку ) `CMake` скриптом ) программы собирается своим собственным `CMake` скриптом, который вызывается из корневого.
### Legacy approach
Весь проект собирается в корневом `CMake`, т.е. все вложенные папки с исходниками собираются не посредственно в корневом скрипте.
